# Flutter Pokédex
Testing Gemini AI power by creating a Pokémon Pokédex using Flutter and Gemini.

## INFO
- Clone this repo
- Edit lib/services/gemini.dart and add you Gemini API Key (line 15)
- App tested on iOS and Android
